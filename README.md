## 💻 Projeto

Este projeto está em construção.. 🚧👷‍♂️

Uma plataforma que mostra ao usuário vagas de emprego contendo as informações do nome da vaga e empresa, localização da vaga, data de postagem que são
mostrados na página principal e na página da vaga será mostrado a descrição da vaga e o email/link no qual o candidato poderá se candidatar 😄.

## 😎 Iniciando 

📖 Para rodar o projeto localmente você precisará seguir algumas instruções.

- É necessário que o <a href="https://nodejs.org/en/">NodeJS</a> esteja instalado em sua máquina.
- É necessário utilizar <a href="https://classic.yarnpkg.com/en/">Yarn</a> ou <a href="https://www.npmjs.com/">Npm</a> para a instalação de pacotes.

### Instalação

```bash
# Clone o repositório
git clone https://github.com/larissadantier/QualyJobs.git

# Instale todos os pacotes

- Npm
npm install

- Yarn
yarn install

```
